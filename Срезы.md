``` Python
a = [1, 3, 8, 7]

print(a[:]) # Выведи мне
>>>[1, 3, 8, 7]

print(a[1:]) # Начиная все с второго элемента
>>>[3, 8, 7]

print(a[:3]) # До третьего элемента
>>>[1, 3, 8]

print(a[::2]) # Каждый второй
[1, 8]
```

# Reverse списка
# [::-1]

``` Python
lst = ["f", 1, 2, 3, 4, 5]

print(lst[::-1])

>>>[5, 4, 3, 2, 1, 'f']
```
